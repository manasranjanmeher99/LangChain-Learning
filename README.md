# LangChain-Learning

**LangChain-Learning** is a beginner-friendly repository that demonstrates how to build AI-powered applications using **LangChain** and modern Large Language Models (LLMs). This project includes hands-on examples with **OpenAI** and **Google Gemini**, covering chat models, prompt engineering, text generation, and language translation.

The repository is organized into simple, well-structured examples to help developers understand LangChain fundamentals before moving on to advanced topics such as Retrieval-Augmented Generation (RAG), AI agents, memory, tools, and workflows.

## 🚀 Features

* 🤖 OpenAI and Google Gemini integration
* 💬 Chat model examples
* ✍️ Text generation using LangChain
* 🌍 Language translation examples
* 📝 Prompt engineering basics
* 🔐 Secure API key management using `.env`
* 📚 Clean project structure for learning and experimentation
* 📖 Easy-to-understand code with notebooks and Python scripts

## 🛠️ Tech Stack

* Python
* LangChain
* OpenAI API
* Google Gemini API
* python-dotenv
* Jupyter Notebook

## 🎯 Purpose

This repository serves as a personal learning journey into LangChain and Generative AI while providing practical examples for students, developers, and AI enthusiasts. It will continue to grow with more advanced implementations, including RAG systems, vector databases, AI agents, memory, and end-to-end LLM applications.
